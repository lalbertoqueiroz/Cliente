Código do beneficiário: também chamado de "código do cliente", "código do cedente", "convênio", é um número/código adotado pelo banco para identificar o contrato firmado para a prestação de um tipo de serviço entre o banco e a empresa cliente (beneficiário).

Este número é obrigatório para a emissão de boletos e possui o seguinte formato: ddddddd, 7 dígitos (6 dígitos + dígito verificador), onde (d) é um número de zero a nove.

As agências (locais físicos de atendimento) são, normalmente, identificadas por um número e um dígito verificador no formato dddd-d, onde (d) é um número de zero a nove. Porém, pode ocorrer de alguma agência não possuir o dígito verificador ou não estar informado em nenhum documento.

Nesse caso, para esse cadastro, pode-se usar o dígito verificador zero, mas é recomendado que você entre em contato com o gerente ou o responsável por sua conta bancária para obter o dígito correto.

Observação: o número da agência não é utilizado durante a emissão do boleto, mas ele é necessário para o processamento dos arquivos CNAB (remessa e retorno).

Quando desmarcado, no boleto o campo Aceite é preenchido pela letra "N" (NÃO), que é a definição padrão.

O exemplo abaixo mostra em destaque esse campo em um boleto:


Quando marcado, no boleto o campo Aceite é preenchido pela letra "S" (SIM), que é usada para quando há necessidade de protesto, indicando se o pagador reconheceu ou não o título como sendo dele (contra ele).

Esse campo indica se o pagador assinou algum documento que deixe claro que ele já tem conhecimento da cobrança. O normal é utilizar "N", pois nesse caso não é necessário a autorização do pagador para protestar tal título.

Caso use "S", o beneficiário precisará de algum documento onde o pagador reconhece a dívida, para então, poder protestar o título em caso de não pagamento.

O campo multa corresponde a porcentagem do valor do boleto que deverá ser cobrada caso o boleto seja pago após o vencimento. Definindo este campo com zero não há cobrança de multa.

Caso o percentual (%) de multa seja informado, após vencido, o boleto terá o valor da multa calculado e informado no campo de instruções do boleto.

Ao emitir um boleto, o emissor pode colocar instruções dirigidas ao caixa ou pagador do boleto no ato do pagamento, como por exemplo: "não receber após o vencimento"; informação do cálculo de multa e juros após o vencimento, etc.

Utilize esse campo para escrever uma instrução padrão que será colocada em todos os boletos emitidos por esta conta bancária. Esse campo se refere a uma linha na área de instruções ao caixa de cada boleto emitido.

Observação: as instruções referentes a multa e juros já são colocadas automaticamente pelo sistema. Portanto, nesse campo deve-se usar uma instrução com outro propósito.

O exemplo abaixo mostra como a instrução padrão é usada juntamente com as informações de multa e juros,
instrução padrão: (Sr. caixa, verifique o vencimento e as taxas de multa e juros antes do pagamento)

Esse campo define em quantos dias após o vencimento um boleto não pago deve ter as taxas de multa e juros aplicadas. Ou seja, com quantos dias um boleto em atraso deve cobrar multa e juros.

Para um boleto que vence hoje, se forem definidas taxas de multa e/ou juros para esse boleto, isso significa que você deseja que essas taxas sejam cobradas, mas a partir de quando? Veja como definir o dia em que as taxas devem ser cobradas para um boleto que vence hoje por exemplo:

  • Cobrar imediatamente após o vencimento (amanhã já cobro encargos) → definir esse campo com o valor 0 (zero);
  • Cobrar somente depois de 1 dia (amanhã não precisa cobrar encargos) → definir esse campo com o valor 1 (um);
  • Cobrar somente depois de 2 dias (amanhã e depois ainda não cobro encargos) → definir esse campo com o valor 2 (dois);
  • Etc.

Ou seja, essas taxas podem ser aplicadas no dia seguinte após o vencimento ou alguns dias depois, esse campo deve informar quantos dias após o vencimento do boleto essas taxas devem ser usadas para recalcular o valor do boleto sobre o atraso no pagamento.
